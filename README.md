# Electron-16-Study

Electron 16 学习

## Init

```
npm init
npm i -D electron
```

## 下载版本说明

- 方法一：
    - 使用不同的镜像：
        ```
        ELECTRON_MIRROR=https://npm.taobao.org/mirrors/electron/
        ELECTRON_MIRROR=https://cdn.npm.taobao.org/dist/electron/
        ```
- 方法二：
    - 不下载二进制文件：
        ```
        ELECTRON_SKIP_BINARY_DOWNLOAD=1
        ```
- 方法三：
    - 使用固定版本
